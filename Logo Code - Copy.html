Method 1: Screenshot (Quick & Easy)

  1. Right-click on the logo variation you want
  2. Select "Inspect" or "Inspect Element"
  3. Right-click on the highlighted element in DevTools
  4. Click "Capture node screenshot"
  5. The image will download automatically

Method 2: Design Software (Professional)

  1. Use the color codes below in your design software (Figma, Adobe Illustrator, Canva, etc.)
  2. Create the bars with these specifications:
    • Bar 1: 60% height, Navy (#eef0f6 on dark / #0b1120 on light)
    • Bar 2: 100% height, Gold (#c9a84c)
    • Bar 3: 40% height, Navy (#eef0f6 on dark / #0b1120 on light)
    • Bar 4: 80% height, Gold Bright (#debb5c)
  3. Add text using Playfair Display Bold font

Color Codes:

  • Navy Dark: #0b1120
  • Navy Light (on dark): #eef0f6
  • Gold: #c9a84c
  • Gold Bright: #debb5c
